## Raffle Prize Picker
A simple CLI raffle tool that randomly selects three winners
and assigns prizes, marking the last one as the Grand Prize.

---
### Features
- Input validation
- No duplicate winners
- Modular structure
- Clean terminal output

---

### Requirements
- **Python 3.8+**
- No external dependencies.

---

### Usage
python raffle_prize_picker.py
