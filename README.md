## Raffle Prize Picker
A simple CLI raffle tool that randomly selects three winners
and assigns prizes, marking the last one as the Grand Prize.

---
### Features
- Collect participant names and prizes
- Pick 3 random winners with no duplicates 
- Save results to a text file

---

### Requirements
- **Python 3.9+**
- No external dependencies.

---

### Usage
python raffle_prize_picker.py
